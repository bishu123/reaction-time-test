# Reaction Time Test ⚡️

A simple, premium Reaction Time Test game built with Vanilla JS.

## Features
-   **Super Simple**: No bloat, just the game.
-   **Premium Design**: Dark mode, glassmorphism, and smooth animations.
-   **Leaderboard**: Tracks your top 5 best scores (saved locally).
-   **Responsive**: Works on desktop and mobile.
-   **Offline Ready**: Zero dependencies, runs directly in the browser.

## How to Play
1.  Click anywhere to start.
2.  Wait for the **Red** box to turn **Green**.
3.  Click as fast as you can.
4.  See if you can get into the "Inhuman reflexes" tier (<150ms)!

## Deployment
This project is ready for **GitHub Pages**.

1.  Push this repository to GitHub.
2.  Go to Settings -> Pages.
3.  Select `main` branch and `/root` folder.
4.  Save.
